omniauth-smailex
================

Connect to your Smailex account using OmniAuth

## Installation

Add this line to your application's Gemfile:

    gem 'omniauth-smailex'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install omniauth-smailex
